# Colors

Colors are an essential aspect to achieve unique and attractive user interfaces.
Tailux provides a wide range of colors to choose from, and we have organized
them into two categories: Common Colors and Semantic Colors.

## Common Colors

Common colors are the fundamental colors used in most user interfaces. They are
used to create primary, secondary, and other colour schemes. Each colour is
represented in 27 different shades, enabling the creation of components that
meet Web Content Accessibility Guidelines (WCAG) standards.

## Semantic Colors

Semantic colors are dynamic and adjust according to the chosen theme, making it
easier to create applications with light and dark modes. For example, if the
primary color is <code>sapphire</code>, to set up a dynamic color background in
Tailwind, you would do something like this:

<Source
  code='<div className="bg-sapphire-500 dark:bg-sapphire-700" />'
  language="jsx"
/>

However, when using the semantic color "primary", like this:

<Source code='<div className="bg-primary-400" />' language="jsx" />

It will automatically adapt to the current theme.
